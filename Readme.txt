Шрифты по ГОСТ 2.304 для использования в Eplan.
http://asutpforum.ru/viewtopic.php?t=3049&p=63162#p63162
Из плюшек:
Пожертвовав middleDot, сделал неразрывный пробел. Ввод: Alt+250
Средние и длинные тире настоятельно рекомендуются вместо минуса во многих случаях. Ввод: Alt+0150, Alt+0151
Устанавливается сразу семейство из четырех шрифтов.
Normal и Normal Italic соответствует ГОСТ 2.304 Type A
Bold и Bold Italic соответствует ГОСТ 2.304 Type B
В сформированном PDF текст не искажается при копировании в буфер обмена.
